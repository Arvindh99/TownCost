### TownCost

Live Website: https://towncost.in/

#### Summary:
TownCost is a community-driven cost-of-living tracking web application that helps users track their personal expenses, understand average cost of 
living by country, state and city based on real user-submitted data. The platform aggregates anonymized expense data and provides insights only 
when enough users contribute, ensuring privacy and reliability.

#### Tech Stack: 
React, Vite, TypeScript, Shadcn UI, Tailwind CSS, Supabase, Vercel, Hostinger

#### How-to Guide
- Sign up or log in
- Add your personal expenses under predefined categories
- Expenses are saved privately in your account
- Once at least 2 users submit data for the same location, community insights become visible
- View cost-of-living averages by: Country, State, City
